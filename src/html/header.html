TmDropdown

by Tanuel

TmDropdown is a JavaScript plugin that allows easy styling of select elements by mimicking their functionality, but providing more freedom

This Project is still in the early stages of development, so bugs may happen.

Download here or view the source on Github

Also available on npm: npm install tmdropdown

Or bower: bower install tmdropdown

Features

  • Allows advanced styling of Select Dropdowns
  • Supports tabbing through Forms (Try pressing tab a few times). Tabindex will be taken over from the original select element
  • Supports navigating through the dropdown via keyboard - up, down, enter, escape
  • Supports disabled options and multi-selects
  • Offers jQuery-Integration, but also works without it
  • You can set a default configuration in the global object TmDropdownConfig like
    • TmDropdownConfig.width = "100%"
    • TmDropdownConfig.emptyText = "No Options available."
